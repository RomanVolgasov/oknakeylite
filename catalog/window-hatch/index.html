Технические характеристики

Кровельные люки Keylite предназначены для использования в неотапливаемых чердачных помещениях.

Оклад из алюминия толщиной 0,6 мм, позволяет устанавливать кровельные люки в крыше в сочетании с любыми видами кровельного материала.

Кровельные люки Keylite открываются по верхней оси, обладают куполом толщиной 3 мм, изготовленным из прозрачного поликарбоната высокой прочности.

Рама сделана из высококачественной древесины.

Благодаря использованию петель оригинальной конструкции, кровельный люк можно плотно закрыть или открыть для вентиляции.
